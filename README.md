# PingPong-Unity

## Controls for Player1:
W = Move up <br />
A = Move left <br />
S = Move down <br />
D = Move right <br /> 

## Controls for Player2:
Up Arrow = Move Left <br />
Left Arrow = Move Left <br />
Down Arrow = Move Down <br />
Right Arrow = Move Right <br />

>[!IMPORTANT]
>If the scene is empty go to Assets > Scenes > double click PingPong.
